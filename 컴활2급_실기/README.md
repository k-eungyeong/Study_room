# 컴활2급_실기

## 학습 내용

* 엑셀 기본 작업 
   - 셀서식, 고급필터, 조건부서식, 텍스트나누기
* 엑셀 계산 작업
   - 논리함수, 날짜와 시간 함수, 문자열 함수, 수학과 삼각 함수, 데이터베이스 함수, 찾기와 참조 함수, 통계 함수
* 엑셀 분석 작업
   - 정렬, 부분합, 목표값 찾기, 데이터표, 통합, 시나리오, 피벗테이블
* 엑셀 기타 작업
   - 매크로, 차트
