# 엑셀 계산작업
  * 계산식은 항상 "="부터 시작
  * Ctrl + ~ 어떤 수식 사용했는지 확인 가능
  * F4  절대참조($)
    
#### 논리함수
     * =SUM(범위)      합계
     * =AVERAGE(범위)  평균
     * =IF(조건, 참, 거짓) 조건 결과에 따른 결과값
     * =AND(조건1, 조건2, ..)  모두 참인 조건으로 구성
     * =OR(조건1, 조건2, ..)   조건 중 하나라도 참이면 가능
     * =IFERROR(값, 오류값)    값을 표시해 주소 오류시 오류값
     * =IFS(조건1, 값1, 조건2, 값2, .., TRUE, 나머지값)  조건N을 만족하면 값N 표시, 나머지(TRUE)는 나머지값 표시
     * =SWITCH(식, 값1, 결과1, 값2, 결과2, .., 나머지값)  식의 값N이면 결과N 표시, 나머지는 나머지값 표시
     
#### 날짜와 시간함수
     * =NOW() 현재 시스템의 날짜, 시간 표시   - 인수X
     * =TODAY() 현재 시스템의 날짜만 표시     - 인수X
     * =YEAR(날짜) 현재 시스템의 년도만 추출 
     * =MONTH(날짜) 현재 시스템의 월만 추출
     * =DAY(날짜)   현재 시스템의 일만 추출
     * =DATE(년, 월, 일)  입력된 년, 월, 일을 날짜로 만들어줌
     * =HOUR(시간) 시만 추출
     * =MINUTE(시간) 분만 추출
     * =SECOND(시간) 초만 추출
     * =TIME(시, 분, 초) 입력된 시, 분, 초를 시간으로 만들어줌
     * =WEEKDAY(날짜, 1) 요일을 숫자로 표시    - 1은 디폴트라 생략가능/ 일요일 기준(!) 시작 ~ 토요일(7)
     * =WORKDAY(날짜, K) 날짜에서 K일을 더한 날(토, 일 빼고)
     * =EDATE(날짜, K)  날짜의 월에 K를 더한 날
     
#### 문자열 함수
     * =LEFT(문자, K) 문자를 왼쪽에서부터 K번째까지 표시
     * =RIGHT(문자, K) 문자를 오른쪽에서부터 K번째까지 표시
     * =MID(문자, 시작위치, K) 문자를 시작위치서부터 K번째까지 표시
     * =LEN(문자) 문자 길이를 숫자로 표시
     * =UPPER(영어)  모두 대문자로 추출
     * =LOWER(영어)  모두 소문자로 추출
     * =PROPER(영어)  단어의 첫 글자만 대문자로 표시
     * =FIND("찾을문자","문자") 문자에서 찾을 문자의 위치를 숫자로 표시(대소문자 구별)
     * =SEARCH("찾을문자","문자") 문자에서 찾을 문자의 위치를 숫자로 표시(대소문자 구별X)
     * =TRIM(문자) 문자 앞/뒤 공백 제거, 중간은 1칸으로 만들기
     * & : 식끼리, 문자-식, 문자끼리 이어줄 때 사용
   
#### 수학과 삼각함수
     * =ABS(숫자) 숫자를 절대값으로 만듦
     * =INT(숫자) 가장 낮은 정수로 표시
     * =MOD(숫자, K) 숫자/K 해서 나온 나머지 표시
     * =POWER(숫자, K) 숫자^K
     * =ROUND(숫자, K) 숫자를 K자리수까지 반올림해서 표시
     * =ROUNDUP(숫자, K)   -> 올림
     * =ROUNDDOWN(숫자, K)  -> 내림
     * =TRUNC(숫자, 0)      -> ROUNDDOWN과 똑같지만 차이점으로는 0 생략 가능
     * =SUM(범위) GKQRp
     * =SUMIF(범위, "조건", 합계범위)  조건을 만족하는 합계
     * =SUMISF(합계범위, 범위1, 조건1, 범위2, 조건2, ..)  조건들을 만족하는 합계
   
#### 데이터베이스 함수
     * DB 함수(표범위, 열제목, 조건) << 동일하게 적용
       : 표범위에서 조건에 만족하는 열제목의 DB함수 계산 
         1. 같은 행에 조건 2개 이상: AND 조건
         2. 다른 행에 조건 2개 이상: OR 조건
     * =DCOUNT( ~ ) 표범위에서 조건에 만족하는 열제목 개수 - 열제목 아래 데이터가 숫자, 날짜, 시간일 때만 계산
     * =DCOUNTA( ~ ) 위와 같지만, 문자인 데이터를 계산할 때 사용
     * =DMAX( ~ ) 표범위에서 조건에 만족하는 열제목의 최대값
     
#### 찾기와 참조함수
     * =VLOOKUP(기준셀, 참조범위, 열번호, 속성(0 또는 1))  기준셀에 참조되는 값을 참조범위에 있는 열번호에서 찾아주는 것
       - 기준셀에 있는 값이 참조범위에 있는 첫 번째 열에 모두 있다면 속성 정확한 값(0) 찾기
       - 참초범위는 컬럼값 포함X 
     * =HLOOKUP(기준셀, 참조범위, 행번호, 속성) 기준셀에 참조되는 값을 참조범위 내에 있는 행번호에서 찾아주는 것
     * =MATCH(찾을 값, 범위, 0) 찾을 값이 범위에서 몇 번째 행, 열에 있나
     * =INDEX(범위, 행, 열) 범위에서 행과 열만큼 이동해서 값ㅊ ㅏㅈ기
     * =CHOOSE(K, 값1, 값2, ..) K가 1이면 값1, 2면 값2, ..
     
#### 통계함수
     * =AVERAGE(범위) 평균
     * =RANK.EQ(기준셀, 범위, 0(내림차순) 또는 1(오름차순)) 범위에서 기순셀의 순위
     * =COUNT(범위) 숫자, 날짜, 시간의 개수   - 문자X
     * =MAX(범위) 범위의 최대값
     * =MIN(범위) 범위의 최소값
     * =LARGE(범위, K) 범위에서 K번째로 큰 값
     * =SMALL(범위, K) 범위에서 K번째로 작은 값
     * =MEDIAN(범위) 범위의 중간 값
     * =STDEV.S(범위) 표준편차
     * =VAR.S(범위) 분산
     * =COUNTIF(범위, "조건") 범위에서 조건을 만족하는 개수
     * =COUNTISF(범위1, 조건1, 범위2, 조건2, ..) 범위1에서 조건1 만족, .. 개수 
     * =AVERAGEIF(범위, 조건, 평균범위) 범위에서 조건에 만족하는 것을 찾아 평균 범위의 평균내기
     * =AVERAGEIFS(평균범위, 범위1, 조건1, 범위2, 조건2, ..) 범위1에서 조건1을 만족하고,.. 만족하는 평균범위의 평균
     * =COUNTA(범위) 빈 셀을 뺀 모든 셀의 수
     * =COUNTBLANK(범위) 빈 셀 개수
     * =MODE.SNGL(범위) 범위의 최빈수
